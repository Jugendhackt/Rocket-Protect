Rocket-Protect

Was ist Rocket-Protect?

Rocket-Protect weist auf die Verletzlichkeit der eigenen Daten hin.

Obwohl Javascript überall als Einfallstor für Spyware und anderen Viren bekannt ist, schalten nur wenige das Feature ab.

Noch weniger Leute benutzen NoScript.

Rocket-Protect listet alle Daten auf, die man durch JavaScript auslesen kann.

Aufgeführt sind unter anderem die öffentliche IP-Adresse des Computers und dessen Betriebssystem.

Außerdem gibt es für jedes Betriebssystem eine "Mini-Instruction-Guide",

wie man mit bordeigenen Mitteln oder Zusatzprogrammen selber eine sichere VPN-Verbindung herstellen kann.

Des weiteren werden bekannte Server aufgelistet, die den eigenen Internetverkehr anonymisieren.
